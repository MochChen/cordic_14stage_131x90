# cordic_14stage_131x90
资源紧张下实现的角度计算（炫一下技能），供学习参考（个人认为越是极简，学习到的东西的可能性越多）。


# 14-Stage Vectoring CORDIC（14 时钟出角度+模长）
- 纯向量模式，范围 -90° ~ +90°
- 角度缩放：90° = 11790（131×90），输出直接为 度数×131（免除法）
- 模长补偿：1/K = 9949/16384，手推误差 < 10⁻⁵
- 无流水线，14 级裸迭代，< 300 LUT
- 输入 ±32768（15bit 有符号），精度 < 0.01°
- 系数为 Python 手算 + 验证
